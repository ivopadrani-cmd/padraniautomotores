# Padrani Automotores

Aplicación de gestión para concesionaria de automotores.

Aplicación Vite+React con backend local simulado usando localStorage.

## Running the app

```bash
npm install
npm run dev
```

## Building the app

```bash
npm run build
```

## Características

- Gestión de vehículos
- CRM y gestión de clientes
- Ventas y reservas
- Documentos y contratos
- Tareas y eventos
- Peritajes
